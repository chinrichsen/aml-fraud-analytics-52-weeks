# 🚨 AML + Fraud Analytics: 52 Weeks of Publications, Code & Insights 📊🔍

Welcome to my 52-week analytics series focused on the intersection of **AML**, **financial crime**, and **fraud analytics**.

Each week, I’ll review **2 publications**:
- **Monday** → one publication review
- **Thursday** → one publication review

Each post will also be supported by a **GitHub repository folder** containing the files, data, notebooks, models, or analysis artifacts related to that week’s topic. 💻

This project is designed to explore the full spectrum of financial crime analytics — from traditional monitoring logic and risk scoring to advanced techniques like graph analysis, machine learning, and emerging AI-based approaches.

---

## 🎯 What this project is about

This repository is my public learning, thinking, and building space for topics including:

- **Transaction Monitoring**
- **Customer Risk Rating (CRR)**
- **KYC / CDD / EDD**
- **Typologies and red flags**
- **Fraud analytics**
- **Behavioral modeling**
- **Network / graph analysis**
- **Machine learning applications**
- **Model governance and tuning**
- **Alert triage and operational effectiveness**

The goal is not just to summarize articles — it is to connect the ideas to real analytics work, practical implementation, and business impact.

---

## 🗓️ How the 52 weeks are structured

The series is organized into **weekly themes**:

- **Odd weeks** → AML / Financial Crime Analytics
- **Even weeks** → Fraud Analytics

Each week contains:
- **Monday post**: first publication review
- **Thursday post**: second publication review
- **Weekly repository folder**: code, data, notebooks, diagrams, or supporting material

## 💡 Why this project matters

AML and fraud analytics often rely on the same underlying data, tools, and analytical thinking — but they solve different problems.

- **AML** is often about identifying suspicious patterns and supporting regulatory reporting.
- **Fraud** is often about stopping losses in real time.

This series is meant to show both worlds clearly, with practical analytics and thoughtful commentary.
